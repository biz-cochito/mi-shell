# omarchy quickshell

This project provides a set of small, useful, stylish quickshell widgets for use on hyprland/omarchy.

## Main widgets

### `StatusBar/Panel`

Similar to omarchy's default waybar configuration, provides a thin, minimally intrusive bar at the top of the screen containing buttons to toggle menus, a calendar, system tray, workspace switcher, etc.

### `LeftPanel`

Slides into view from the left side of the screen when the toggle button on the status bar is clicked or when the hotkey is invoked. Contains settings and quick configurations for hyprland/omarchy.

### `RightPanel`

Likewise hidden until toggled but comes into view from the right. Contains a simple LLM chat window (must support Google/Antigravity/Gemini), a bookmark manager, and file browser. A tab bar at the top of the panel lets the user switch between these functions.
